機能の概要と導線

テーピング管理機能は、練習・試合の前にメンバーが希望部位を申請し、トレーナーが準備・費用集計・在庫管理を一元化できる機能です。

目的内容
需要把握何人がどの部位を希望するかをイベント前に把握し、トレーナーの準備・人数調整に活用
在庫管理今後のイベントで必要なテープ種別・本数を集計し、不足を事前検知
費用請求暦年（1月1日〜12月31日）のメンバー別費用を自動集計し、期末請求に活用

アクセス導線

画面上部ナビゲーションの 「Taping」 をタップすると、年度集計・在庫確認ページ（/taping）が開きます。

イベント詳細からの導線：各イベント詳細ページのタイトル横に小さな「テーピング確認」ボタンがあります。そこからイベント別一覧 → リクエスト申請と進めるのが最も自然な動線です。
2

テーピングリクエストの申請 全メンバー

イベント前日までに希望部位を申請します。/taping/request から直接、またはイベント別一覧画面下部の「テーピングリクエストをする」ボタンから遷移します。

テーピングリクエスト申請フォーム（/taping/request）

申請手順

  1. 対象イベントを選択：直近40日以降のイベントが新しい順で表示される。最新イベントが自動選択される

  2. 部位にチェック：テーピングが必要な部位にチェックを入れる（複数選択可）。単価の目安が右に表示される

  3. 「送信する」をタップ：「送信しました」メッセージが表示されれば完了

申請の変更：同じイベントを選んで再送信すると、前回の申請が上書きされます。再送信時は前回の選択が自動で読み込まれた状態で表示されます。

注意：送信ボタンは1部位以上選択しないと押せません。申請の取り消しは現在のUIでは行えません（0件選択での送信不可）。
イベント別一覧からの申請：イベント詳細 →「テーピング確認」→ 下部「テーピングリクエストをする」と進むと、イベントが自動選択された状態で申請フォームが開きます。
3

イベント別リクエスト確認 全メンバー

イベント詳細画面のタイトル横「テーピング確認」ボタン、または /events/{イベントID}/taping にアクセスすると、そのイベントへの全申請一覧が表示されます（イベント前後いつでも確認可能）。

イベント別テーピングリクエスト一覧

表示情報説明
申請人数そのイベントに申請したメンバー数
合計金額全申請の単価合計（目安）
推定テープ本数全申請の使用テープ本数合計（テープ使用量設定がある場合）
メンバー別内訳申請者の顔写真・名前と選択部位・小計

画面下部の「テーピングリクエストをする」ボタンで、申請フォームへ移動できます（当日その場での追加申請にも使えます）。

4

年度集計・テープ在庫の確認 全メンバー

ナビゲーションの「Taping」をタップすると表示されるページです（/taping）。

年度集計・テープ在庫状況ページ（/taping）

費用集計（暦年累計）

暦年（1月1日〜12月31日）にテーピングリクエストを申請したメンバーの費用を集計します。金額の大きい順に並び、ページ上部に合計金額も表示されます。期末の費用請求にご活用ください。

テープ在庫状況

今後のイベントへのリクエスト申請を集計し、テープ種類ごとの必要本数と基本ストック本数を比較します。

表示意味
必要 ○本今後のイベント申請から算出した推定消費量
ストック ○本テープ素材マスタで設定した基本ストック本数（目標在庫）
⚠ 不足必要本数がストック本数を上回っている（補充が必要）
✓（緑）ストックが十分にある
ストック未設定基本ストック本数が未入力のため比較できない
ストック未設定の場合：「マスタ管理」ボタン（ページ下部）→「テープ素材」タブで基本ストック本数を設定すると在庫アラートが機能します。
5

マスタ管理 — 施術メニュー Admin / Trainer / Staff

「/taping」ページ下部の「マスタ管理」ボタン、または直接 /taping/master にアクセスします。

前提：テープ使用量を設定するには、先に「テープ素材」タブでテープ素材を1件以上登録してください。テープ素材がないと、施術メニューにテープ使用量の入力欄が表示されません。

施術メニュー一覧（施術メニュータブ）

施術メニュー追加・編集モーダル

施術メニューの追加

  1. 「+ 施術メニューを追加」ボタンをタップ → モーダルが表示される

  2. 名称（必須）、単価目安、テープ使用量、備考、表示順を入力する

  3. 「保存する」ボタンをタップ

施術メニューの編集・削除

一覧の各行をタップすると編集モーダルが開きます。削除はモーダル下部の「削除する」から行います。

モーダルを閉じる方法：「✕」ボタン / モーダル外をタップ / Escape キーのいずれでも閉じられます。

各フィールドの説明

フィールド説明
名称 *申請フォームに表示される部位名（例：足首（片足））
単価目安（円）費用集計に使用する1回あたりの目安金額
テープ使用量（本）テープ素材ごとの使用本数目安（0.5本単位で入力可）。在庫集計に使用
備考マスタ管理画面内でのメモ。メンバーの申請フォームには表示されない
表示順申請フォームでの表示順（数字が小さいほど上に表示）
無効にするチェックを入れると申請フォームに表示されなくなる
スナップショット保存について：申請時点の名称・価格・テープ使用量が申請データに保存されます。マスタを後から変更しても、過去の申請データは自動更新されません（費用集計の一貫性を保つための仕様です）。

マスタ管理 — テープ素材 Admin / Trainer / Staff

「テープ素材」タブに切り替えると、テープの種類（ホワイト・キネシオ等）を管理できます。

テープ素材管理（テープ素材タブ）

テープ素材の追加

  1. 「テープ素材」タブをタップして切り替える

  2. 「+ テープ素材を追加」ボタンをタップ → モーダルが表示される

  3. 名称（必須）、基本ストック本数、表示順を入力して「保存する」

テープ素材の編集・削除

一覧の各行をタップすると編集モーダルが開きます。削除はモーダル下部の「削除する」から行います。

各フィールドの説明

フィールド説明
名称 *テープ種類の名前（例：ホワイト、キネシオ、バンテージ）
基本ストック（本）通常時の目標在庫本数（0.5本単位で入力可）。在庫不足アラートの基準値
表示順施術メニューのテープ使用量入力欄での表示順
無効にするチェックを入れると在庫集計の対象から除外される
初期設定の推奨テープ素材：
ホワイト / キネシオ / バンテージ / アンダーラップ / エラスティックテープ

アクセス権限と初期セットアップ

アクセス権限一覧

ページアクセスできる人
/taping/request（申請フォーム）全ログインメンバー
/events/{id}/taping（イベント別一覧）全ログインメンバー
/taping（年度集計・在庫）全ログインメンバー
/taping/master（マスタ管理）Admin / Trainer / Staff のみ

「Trainer」「Staff」は Slack プロフィールの 肩書（Title）フィールドで判定します（大文字小文字不問）。
例: Trainer/OL、Staff、Head Trainer など「trainer」または「staff」を含む文字列。

初期セットアップの手順（Admin / Trainer / Staff 向け）

  1. テープ素材を登録する：「Taping」→「マスタ管理」→「テープ素材」タブ → 使用するテープ種類を追加し、基本ストック本数を設定する

  2. 施術メニューを登録する：「施術メニュー」タブ → 各部位（足首・膝・肩など）を追加し、単価目安とテープ使用量を設定する

  3. メンバーへ周知する：ナビの「Taping」→「テーピングリクエストをする」から申請できることをメンバーに伝える
